# AISight

### Запуск backend
В папке backend
#### 1. Создание окружения
Linux/Mac:
 python3 -m venv venv
 source venv/bin/activate
Windows:
 python3 -m venv venv
 venv\Scripts\activate
#### 2. Установление зависимостей
pip install -r requirements.txt
#### 3. Запуск сервера
uvicorn main:app --reload --port 8000

### Запуск Frontend
В папке frontend

**npm install**
**npm run dev**
